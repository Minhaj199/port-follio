EV Ride Booking Mobile App Web Application React Django

Hound Mobility — Book EV Rides

Hound Mobility is a modern EV ride booking application focused on providing smarter, cleaner, and more sustainable transportation solutions. The app enables users to seamlessly book multiple ride categories including Hourly Rentals, Premium Rides, Airport Transfers, and Outstation Trips through an intuitive and user-friendly mobile interface. The platform was designed with a strong focus on user convenience and eco-conscious mobility. Users can manage their profiles, monitor current trip status in real time, access ride activity and trip history, and receive important ride-related notifications directly within the app. The application also promotes awareness about sustainable transportation by highlighting the environmental benefits of EV rides, including reduced pollution, quieter streets, and cleaner urban travel experiences. Features like transparent pricing, easy booking flow, responsive design, and smooth navigation help create a reliable and premium user experience. Worked collaboratively on developing responsive UI components, improving mobile usability, and creating a clean booking experience optimized for everyday commuters and travelers.
